# SoftwareWorld live profile evidence

## Current registration/claim route

Observed in a user-verified headed Camoufox session on July 23, 2026:

- Public registration fields: **Full Name**, **Business Email Address**, **Password**, **Confirm Password**.
- New software/service listings display a **one-time $99 approval fee** for a lifetime listing.
- The same page says an existing SoftwareWorld profile can be registered and claimed free of cost.
- Vendor login says it manages software/service profiles, rankings, and subscriptions.

These facts can change. Capture the live authenticated profile form and its constraints immediately before use.

## Live public profile surfaces

Product pages can show: logo and verified-profile state; tagline; overview; category-specific descriptions;
features; integrations; pricing; screenshots/YouTube video; vendor/founding/location/employee details; support;
training; licensing/deployment; target company sizes; supported languages; industries; FAQ; reviews; CTA; and
platform-generated review-team assessments, badges, pros/cons, and ranking signals.

category-relevant five peers for each product; this set is evidence of page structure, not a reusable competitor list.
