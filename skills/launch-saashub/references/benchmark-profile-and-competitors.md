# SaaSHub benchmark, profile surfaces, and competitors

## Five-peer benchmark

live alternatives or comparison page too. For each, record category tags, buyer job, description, Website/CTA,
pricing, platforms, release date, features/specs, media, proof, alternatives, comparisons, and gaps.

## Public-profile interpretation

Current pages can show a product statement, rating/review count, pricing, Website, Alternatives, Status Page,
screenshots/images, Features & Specs, badges, comments/discussion, related categories, verified alternatives, and
popular comparisons. Treat reviews, social/community data, popularity signals, badges, and comparative ordering as
platform/community-generated. Never manufacture them or write a description that pretends they are vendor facts.

## Competitor rule

Add only SaaSHub-listed products that offer the same core job to a comparable buyer. Record the shared job, genuine
distinction, listing URL, and evidence. Do not add complementary integrations, unrelated category leaders, or a
product merely because it has search volume.
