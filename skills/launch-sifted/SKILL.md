# Launch on Sifted

## Goal

Create `.ulpi/launch/sifted/` with `ROUTE-AND-FIT.md`, `BENCHMARK.md`, `EU-NEWS-PACKET.md`, `DATA-CORRECTION.md`,
`COMMERCIAL-DISCLOSURE.md`, `FORM-CAPTURE.md`, `FOLLOW-UP-RUNBOOK.md`, `analytics.md`, and shared positioning.

## Workflow

### 1. Select a defensible route

Use sources and live research to return `EDITORIAL PITCH READY`, `DATA CORRECTION READY`, `EVENT/NEWSLETTER ROUTE`,
`COMMERCIAL OPTION TO REVIEW`, or `NOT NEWSWORTHY`.

### 2. Benchmark exactly five peers

Record sector, geography, title, author/date, evidence, access label, narrative shape, sponsor label, and CTA; derive
original positioning without copying.

### 3. Assemble source-governed packets

Prepare official facts, story hook, primary links, approved quote, founder/company context, media, category, correction
contact, and separately labelled commercial/Pro terms. Capture actual gated fields only after approval.

### 4. Execute and measure with approval

Present exact route, assets, form values, terms, spend, disclosure, and owner. Verify live facts/labels/links and track
qualified referrals, source accuracy, publication/data status, and spend separately. End `LIVE — SIFTED RUNBOOK ACTIVE`.
