# Launch on Startups.gallery

## Goal

Create `.ulpi/launch/startups-gallery/` with `ROUTE-AND-FIT.md`, `BENCHMARK.md`, `TEN-WORD-DESCRIPTION.md`,
`SUBMISSION-PACKET.md`, `FORM-CAPTURE.md`, `SOURCE-REGISTER.md`, `analytics.md`, and positioning.

## Workflow

### 1. Check fit and route

Return `CURATED SUBMISSION READY`, `NEEDS CLEARER EARLY-STAGE FIT`, or `NOT A FIT`. Confirm a genuine startup, an
authorized representative, canonical website, coherent product, and a concise factual positioning statement.

### 2. Benchmark exactly five peers

Use `$browse` to inspect exactly five comparable current cards/profiles and nearest category filters. Compare category,
stage, funding source, geography/work type, description construction, and destination URL. Identify the shortest
truthful differentiator for the applicant.

### 3. Build the form-ready packet

Prepare: official name; canonical website; three 10-word description candidates with exact word counts; founder/
employee/VC/other relationship; funding source; submitted-by identity; authorized email; optional referral source;
logo/screenshot only if a live form later requests it; evidence links; and correction owner. Select one conservative,
buyer-readable description rather than marketing slogans.

### 4. Review and submit manually with approval

Show the current form-field map, exact 10-word choice, public facts, email owner, and source register for approval.
The authorized human submits manually. Save the confirmation, check the live card after review, request corrections
with evidence, and measure inclusion, qualified visits, and attributed referrals. End `LIVE — STARTUPS.GALLERY RUNBOOK ACTIVE`.
