# Crozdesk profile and vendor-route evidence

Current public Crozdesk materials state that software vendors may apply for a free listing, then obtain vendor-portal
access after application review to complete and submit their listing. The current public claim guidance describes
claiming an existing unowned listing after Crozdesk checks authority. Capture the live vendor application, claim,
and portal fields immediately before use.

Observed public listing surfaces include overview; main and other categories; score/rating/reviews; pricing and plans;
trial/freemium; suitable users; native applications; languages/regions; vendor information; features; integrations;
FAQs; alternatives/similar products; screenshots; user reviews; and last-modified date. Marketplace scores, rankings,
reviews, category ordering, awards, and buyer-match output are not vendor claims.
